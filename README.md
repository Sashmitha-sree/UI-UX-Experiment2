# UI-UX-Experiment2

## Aim:
To design a basic mobile and web layout for an existing case study using Gestalt principles, UI elements, and usability principles.
The project aims to create visually appealing, user-friendly interfaces that enhance interaction and maintain brand consistency using Canva and other design tools.
## Algorithm:
1.Select an existing case study for layout design.


2.Plan and sketch the layout structure for both mobile and web versions.


3.Apply Gestalt principles such as proximity, similarity, and continuity.


4.Add essential UI elements like buttons, icons, and navigation bars.


5.Ensure usability through consistency, readability, and accessibility.


6.Create a brand logo in Canva following color and font guidelines.


7.Combine all design elements to finalize the mobile and web layouts.
